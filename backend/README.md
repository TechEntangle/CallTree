# Backend - Supabase

This directory contains database migrations, edge functions, and backend configurations.

## Structure

- `migrations/` - Database schema migrations
- `edge-functions/` - Supabase Edge Functions
- `types/` - TypeScript type definitions

## Setup

Migrations are managed via Supabase CLI.

## Database Migrations

Run migrations:
```bash
supabase db push
```

Reset database:
```bash
supabase db reset
```
